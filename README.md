# Electron七牛客户端
使用election-vue制作的一个简单的七牛客户端

## 使用
### 下载项目
`git clone git@github.com:TangGe1119/qiniu-client.git`
### 开发预览
`yarn install`
`yarn dev`
### 打包
`yarn build`

## 已有功能
* [x] 通过ak和sk获取到对应的空间
* [x] 设置默认下载路径
* [x] 获取backet内文件
* [x] 图片预览
* [x] 文件的下载（设置的下载路径）、上传、删除、链接复制等
