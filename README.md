# Electron七牛客户端
使用election-vue制作的一个简单的七牛客户端

## 使用
### 下载项目
`git clone git@github.com:TangGe1119/qiniu-client.git`
### 开发预览
`yarn install`

`yarn dev`
### 打包
`yarn build`

## 已有功能
* [x] 通过ak和sk获取到对应的空间
* [x] 设置默认下载路径
* [x] 显示bucket内文件
* [x] 新建、删除bucket
* [x] 图片预览
* [x] 空间域名选择
* [x] 文件的下载（设置的下载路径）、上传、删除、链接复制（自动根据选择的域名生成链接）、重命名等
* [x] 文件默认按时间进行排序
* [x] 文件按照文件大小和上传时间进行升序或降序排序
* [x] 文件列表前缀搜索
